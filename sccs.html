Import our scss

If you want to use our framewok with scss version, you just need to import frameworks.scss file in your scss or css file. After that you can overwrite propety and redfine our style.

We have also some variables than you can redefine but we wil explain after this topic. 


@import '../scss/framework';

body{
    // YOUR RULES !!!
}

Mixin

The both of mixin grid-off() and grid-col() are very interresting, they permit you to add size columns at elements for all screen size.
In the example below i define class using the both of mixin to manage alert size.

The alert size will be :

  • 2 columns offset and 8 columns large for computer screen
  • 3 columns offset and 6 columns large for tablet screen
  • 4 columns offset and 4 columns large for mobile screen

I have special classes to manage ma width =D × 

.my_size_class{
    @include grid-off(2,3,4);
    @include grid-col(8,6,4);
}

 //classe added
<div class="alert alert-error  my_size_class align_center">
   ....
 <a href="" class="alert-close" data-dismiss="alert">×<a/>
</div>
