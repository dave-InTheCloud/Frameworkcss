<head> Marked

Before to add our css or sccs version of the framework, you need to implement head marker.

The code below contains the different important marked :

  • UTF-8 like standard encoding
  • Compatibility for IE8
  • Screen size detection (mobile, tablet, ...)
  • Font import by google


<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1">
<link href="https://fonts.googleapis.com/css?family=Roboto+Condensed" rel="stylesheet" type="text/css">

<!--[if lt IE 9]>
<script src="https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.js"> </script>
<![endif]-->

Grid


Element size

Our framework is compose by a grid of 12 columns. Each element you want to resize need a marked container.
No matter the size of the browser, each of these columns will always have an equal width. You have 3 classes for 3 screens dimension : small, medium and large.

Example :

col-s-6
col-s-3
col-s-3
col-s-8
col-s-4
col-s-10
col-s-2
col-s-12

<div class="row">
    <div class="col-s-6 presentation">SMALL-6</div>
    <div class="col-s-3 presentation">SMALL-3</div>
    <div class="col-s-3 presentation">SMALL-3</div>
</div>
<div class="row">
    <div class="col-s--1 presentation">SMALL-1</div>
    <div class="col-s--2 presentation">SMALL-2</div>
    <div class="col-s--9 presentation">SMALL-9</div>
</div>
<div class="row">
    <div class="col-s-12 presentation">SMALL-12</div>
</div>

Note : If the size classe is not define for a screen size, automatically the lower class (if defined) will be applied.

col-s-12 col-md-6
<div class="small-12 medium-6 presentation">SMALL-12 MEDIUM-6</div>

The element on the top will have a size of 12 columns for the mobile screens, and a size of 6 columns for the computer and tablet screens

Offsets

You can add offset, this offset can be define like columns, so for that use offset classes. In that case also 3 classes exist for each screen size (offset-sm, offset-md, offset-lg).

.col-md-4
.col-md-4 .offset-md-4
.col-md-3 .offset-md-3
.col-md-3 .offset-md-3
.col-md-6 .offset-md-3

<div class="row">
    <div class="col-md-4 presentation">.col-md-4</div>
    <div class="col-md-4 offset-md-4 presentation">.col-md-4 .offset-md-4</div>
</div>
<div class="row">
    <div class="col-md-3 offset-md-3 presentation">.col-md-3 .offset-md-3</div>
    <div class="col-md-3 offset-md-3 presentation">.col-md-3 .offset-md-3</div>
</div>
<div class="row">
    <div class="col-md-6 offset-md-3 presentation">.col-md-6 .offset-md-3</div>
</div>
